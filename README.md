## Proyecto Node y OpenAI - Generador avatares
### 26-Abril-2026

### Ejecutar en dev

1. Clonar proyecto
2. Ejecutar `npm install` en la carpeta raiz del proyecto.
3. Crear `.env` basado en `.env.template`.
4. Ejecutar `npm start` para ejecutar el proyecto.
5. Acceder a la ruta "http://localhost:3000".

_NOTA: Hay que generar una APY Key de una cuenta de OpenAI con créditos disponibles y agregarla al archivo .env para el funcionamiento del proyecto._
